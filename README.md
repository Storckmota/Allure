# Alluré Branding

Site institucional da Alluré Branding, com direçăo visual editorial, motion design e experięncia premium para apresentaçăo de serviços de registro de marcas.

## Stack

- Vite
- HTML
- CSS
- JavaScript
- GSAP

## Comandos

npm install
npm run dev
npm run build
npm run preview

## Deploy

Projeto preparado para deploy na Vercel.

Configuraçăo esperada:

- Framework: Vite

- Output directory: dist
